# Privacy Policy

Chaterm collects anonymous usage data to improve the product. Your code, prompts, and conversations are never collected.

## Overview

Chaterm uses [PostHog](https://posthog.com) (an open-source analytics platform) to collect and analyze anonymous usage data. This feedback loop helps the team understand how developers use the product and guides improvements to functionality and user experience.

Our telemetry implementation is completely transparent -- you can review the [source code](https://github.com/chaterm/Chaterm) to see exactly what is tracked.

::: info Transparency
All collected data is anonymized before it leaves your device and contains no personally identifiable information (PII). You can audit the telemetry implementation at any time in the public repository.
:::

## What We Track

We collect basic anonymous usage data, including:

| Category | Examples |
|----------|----------|
| **Task interactions** | Task start and end times, conversation flow (without content) |
| **Mode and tool usage** | AI modes selected, tools invoked |
| **Token usage** | Conversation-length metrics for cost estimation (no actual token content) |
| **UI activity** | Feature usage across the interface |

## How to Opt Out

Telemetry is completely optional:

- When you first install or update Chaterm, a notice about anonymous telemetry is displayed.
- You can change your preference at any time from the privacy settings page.

For full control over data collection, masking, and sync, see [Privacy Settings](/docs/privacy/settings/).
