# Account Management

Manage your account registration, authentication methods, and security settings.

## Account Registration

### Email Registration

Register an account using your email address. A verification code is sent to your inbox to confirm your identity.

### Third-Party Authentication

Sign in quickly with an existing account from one of the following providers:

| Provider | Description |
|----------|-------------|
| **Google** | Sign in with your Google account |
| **GitHub** | Sign in with your GitHub account |
| **Apple ID** | Sign in with your Apple account |

## Account Information Modification

You can update the following fields from your profile:

- Display name
- Username
- Phone number
- Login password

## Suspicious Login Detection

The system automatically monitors for unusual activity, including:

- Sign-ins from new locations
- Sign-ins from unrecognized devices

When suspicious activity is detected, the system will:

1. Send a security notification to your registered email.
2. Trigger two-factor verification.
3. Require you to confirm your identity before proceeding.

::: warning Security Reminder
If you notice any suspicious login activity, change your password immediately from the [Change Password](/docs/user/edit/) page.
:::
