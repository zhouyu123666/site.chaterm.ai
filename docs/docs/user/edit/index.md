# Change Password

Change your account password to keep your Chaterm account secure.

## Steps

1. Click the **User** icon in the left sidebar to open your profile page.
2. Find and click the **Reset Password** button.
3. Enter your new password in the password input field.
4. Click **Confirm** to save the change.

::: warning Password Requirements
- Password must contain at least **8 characters**.
- Use a combination of letters, numbers, and special characters for stronger security.
- Do not reuse a password you have used before.
:::
