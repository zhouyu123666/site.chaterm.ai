# Profile

View and manage your account details, subscription tier, and security information from your profile page.

## Accessing Your Profile

Click the **User** icon in the left sidebar to open your profile.

## Account Information

### Subscription Tier

Your current plan is displayed at the top of the profile page. Chaterm offers four tiers:

| Tier | Description |
|------|-------------|
| **FREE** | Basic access with limited features |
| **LITE** | Extended features for individual users |
| **PRO** | Full feature set for power users |
| **ENTERPRISE** | Team management and advanced controls |

### Basic Information

| Field | Description |
|-------|-------------|
| **Name** | Your real name, displayed in your profile and shared spaces |
| **Username** | The unique identifier you use to sign in |
| **Email Address** | Used for notifications, verification codes, and account recovery |
| **Phone Number** | Used for security verification and SMS notifications |

### Security Information

| Field | Description |
|-------|-------------|
| **Password** | Your account login password. You can reset it from this page. |
| **IP Address** | The network address of your current login session |
| **MAC Address** | The hardware address of the device you are using |
