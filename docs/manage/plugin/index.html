Extension Management ​

The extension feature allows you to install and manage extension plugins to extend Chaterm's functionality. Through the extension system, you can customize and enhance Chaterm's capabilities according to your personal needs.

Browsing Extensions ​

Click "Extensions" in the left menu to enter the extension management page. The extension management page provides the following features:

  • Extension Marketplace: Browse all available extension plugins
  • Installed Extensions: View and manage installed extensions
  • Search Function: Quickly find needed extensions through keywords

Installing Extensions ​

Installation Steps ​

  1. In the extension management page, browse or search for the extension you need
  2. Click on the extension card to view detailed information, including:
    • Extension detail description
    • Feature description
    • Installation information
  3. Click the "Install" button to start installing the extension
  4. Wait for the installation to complete. Extensions will be automatically enabled after installation

Installation Methods ​

Method One: Install from Extension Marketplace

  • Browse available extensions in the "Extension Marketplace" tab of the extension management page
  • Click the "Install" button of the extension to install

Method Two: Install from Local File

  • Click the "Install from File" button
  • Select the extension package file (.chaterm format)
  • The system will automatically verify and install the extension

Tip

Extensions will be automatically enabled after installation. You can view and manage them in the "Installed Extensions" list.

Managing Extensions ​

In the extension list, you can perform the following operations on installed extensions:

Enable/Disable Extensions ​

  • Enable Extension: Click the "Enable" button of the extension, and the extension will take effect immediately
  • Disable Extension: Click the "Disable" button of the extension, and the extension functionality will be temporarily disabled, but it will not be uninstalled

Notes

Disabling some core extensions may affect Chaterm's normal functionality, please operate with caution.

Configuring Extensions ​

Some extensions support configuration options, allowing you to customize extension behavior:

  1. In the extension list, click the "Configure" button of the extension
  2. Modify extension settings in the configuration dialog
  3. Configuration items may include:
    • Feature switches
    • Display options
    • Behavior parameters
    • Integration settings
  4. Click "Save" to make the configuration take effect

Updating Extensions ​

When a new version of an extension is available:

  • An update notification will be displayed in the extension list
  • Click the "Update" button to upgrade to the latest version
  • Extension configuration is usually preserved after updating

Uninstalling Extensions ​

If you no longer need an extension, you can uninstall it:

  1. In the extension list, find the extension to uninstall
  2. Click the "Uninstall" button
  3. Confirm the uninstall operation
