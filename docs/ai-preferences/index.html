AI Preferences ​

The AI preferences feature allows you to customize the behavior and response patterns of your AI assistant, creating a personalized AI interaction experience.

General Settings ​

1. Extended Thinking ​

  • Status: Enabled by default
  • Function: Set token count to achieve more comprehensive and detailed reasoning with increased budget
  • Description: Improve the quality and accuracy of AI responses by increasing reasoning depth

2. Auto Execution ​

  • Status: Disabled by default
  • Function: When enabled, Agent mode will automatically execute commands without user confirmation
  • Note: It is recommended to enable this feature only after becoming familiar with system operations

3. Security Configuration ​

  • Function: Configure security policies for the AI assistant to prevent execution of dangerous commands
  • Importance: Protect system security and avoid accidental execution of harmful operations

Reasoning Features ​

OpenAI Reasoning Work Level ​

Choose different reasoning intensities to balance performance and cost:

  • Low (default) - Basic reasoning, fast response, lower cost
  • Medium - Balanced reasoning, moderate response time and accuracy
  • High - Deep reasoning, highest accuracy, longer response time

Proxy Configuration ​

Note: Proxy configuration is disabled by default and should only be enabled when needed

Configuration Options ​

  • Protocol: Select proxy protocol type
  • Hostname: Proxy server address
  • Port: Proxy server port
  • Authentication: Proxy account and password settings

Terminal Settings ​

Shell Integration ​

  • Function: Configure Shell integration timeout
  • Default Value: 4 seconds
  • Description: Set timeout for terminal command execution to prevent long-running unresponsive commands from blocking the system
