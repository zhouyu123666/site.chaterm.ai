AI Preferences ​

The AI preferences feature allows you to customize the behavior and response style of the AI assistant, creating a personalized AI interaction experience.

General Settings ​

1. Extended Thinking ​

  • Status: Enabled by default
  • Function: Set the number of tokens, increasing the budget enables more comprehensive and detailed reasoning
  • Description: Improve the quality and accuracy of AI responses by increasing reasoning depth

2. Auto Execute ​

  • Status: Disabled by default
  • Function: When enabled, Agent mode will automatically execute commands without user confirmation
  • Note: It is recommended to enable this feature after familiarizing yourself with system operations

3. Security Configuration ​

  • Function: Used to configure AI assistant security policies to prevent execution of dangerous commands
  • Importance: Protect system security and avoid accidentally executing harmful operations

Reasoning Features ​

OpenAI Reasoning Work Level ​

Choose different reasoning intensities to balance performance and cost:

  • Low (default) - Basic reasoning, fast response, lower cost
  • Medium - Balanced reasoning, moderate response time and accuracy
  • High - Deep reasoning, highest accuracy, longer response time

Proxy Configuration ​

Note: Proxy configuration is disabled by default, enable only when needed

Configuration Options ​

  • Protocol: Select proxy protocol type
  • Hostname: Proxy server address
  • Port: Proxy server port
  • Authentication: Proxy account and password settings

Terminal Settings ​

Shell Integration ​

  • Function: Configure Shell integration timeout
  • Default Value: 4 seconds
  • Description: Set the timeout for terminal command execution to prevent long-running unresponsive commands from blocking the system
