Mobile Quick Start ​

This guide walks you through installing the mobile app, signing in, adding your first host, opening an SSH session, and trying AI features.

Step 1: Install the App ​

iOS ​

  1. Open App Store
  2. Search for and install Chaterm
  3. Complete the permission prompts when you launch it for the first time

Android ​

  1. Open Google Play
  2. Search for and install Chaterm
  3. Complete the permission prompts when you launch it for the first time

Tip

If you want data sync and full AI features, it is best to sign in before you start using the app.

Step 2: Sign In ​

Chaterm mobile supports both full sign-in usage and browsing first before finishing sign-in later.

  • Email verification code sign-in
  • Username and password sign-in
  • Third-party sign-in, depending on the options currently available in the app

Step 3: Add Your First Host ​

  1. Open the Assets page
  2. Enter Direct Connections and tap the + button in the lower-right corner
  3. Choose New
  4. Fill in the host label, IP address, port, username, and other details
  5. Select password authentication or key authentication
  6. Tap Save

If you have enabled data sync on desktop, your asset data is encrypted and synced to mobile automatically.

For more details, see Asset Management.

Step 4: Open an SSH Connection ​

  1. Tap the target host card in the asset list
  2. Chaterm automatically creates the SSH connection
  3. After the connection succeeds, you enter the terminal screen

Common actions:

  • Tap a host card to connect directly
  • Long-press a card to edit, favorite, or delete it
  • Use search to quickly filter by host name, IP, username, or group

Step 5: Start Using AI ​

Mobile provides two main interaction modes:

  • Term mode: directly runs the command you enter
  • Chat mode: lets AI understand the request and suggest commands or actions

Basic flow:

  1. Open the AI dialog page
  2. Connect to a target host or select a model
  3. Switch modes from the selector at the bottom of the input box
  4. Enter a question or command, or use voice input
  5. Review the execution result or the AI response

For more details, see AI Dialog.

Recommended First Run ​

  1. Add a test host
  2. Start one SSH session
  3. Run pwd or uname -a in Term mode
  4. Switch to Chat mode and ask one Linux command question
  5. Open Profile and configure theme, language, and AI preferences
